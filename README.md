# Kuickart Task Manager

Frontend-only task manager built with Vite + React JSX.

## Roles
- Owners: Germany, France, Pakistan
- Admins: Germany, France, Pakistan
- Users: country-based operational users

## Workflow
- Admins and owners create tasks
- Users view their assigned tasks
- Users submit completed work for approval
- Admins or owners approve or reject

## Storage
This version is fully frontend-only and stores demo data in `localStorage`.
That makes it suitable for Netlify deployment, but not secure for production.

## Commands
```bash
npm install
npm run dev
npm run build
```

## Deployment
This app can be deployed directly to Netlify as a static frontend.
